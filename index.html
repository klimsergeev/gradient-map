Как работают цветовые гармонии

Оригинал — исходный рыже-коричневый градиент полосок, без перекраски. Режим по умолчанию.

Аналоговая — для каждой полоски её базовый цвет сдвигается на ±25° по кругу hue, давая три родственных оттенка. Мягко, не контрастно, сохраняет общее настроение картинки.

Триада — классическая триада 120°: базовый цвет плюс два спутника, разнесённых на 120° по кругу. Максимальный контраст.

Комплиментарная — база + противоположный цвет через 180° по кругу hue. Максимальный двуцветный контраст.

Монохромная — один hue, варьируется только светлота и слегка насыщенность. Тёмные стопы менее насыщенные — как тени в реальности. Самая сдержанная гармония.

Квадратичная — 4 цвета через 90° (квадрат на цветовом круге). Для каждой полоски берутся 3 подряд вершины квадрата.

Разделённая комплиментарная — вместо точного комплемента берутся два соседних к нему цвета (±30° от 180°). Мягче, чем чистый комплемент, но с сохранением контрастности.
